Open Source at Datacratic

At Datacratic we love open source and do everything that we can with it, including contributing back.

JML

"Jeremy's Machine Learning Library" contains many of our core machine learning algorithms.
https://github.com/datacratic/jml

StarCluster

StarCluster is a utility for creating and managing computing clusters hosted on Amazon's Elastic Compute Cloud (EC2).
http://star.mit.edu/cluster/

Visualize

Datacratic's Dataviz Framework.
https://github.com/datacratic/visualize

Data Projector

Visualizing High-Dimensional Data in the Browser with SVD, t-SNE and Three.js.
https://github.com/datacratic/data-projector

RTBkit

RTBkit is a real-time bidder framework designed for use on advertising exchanges such as the Rubicon Project, Nexage, App Nexus, The Google Ad Exchange and others.
http://rtbkit.org

PivotTable.js

Javascript Pivot Table (aka Pivot Grid, Pivot Chart, Cross-Tab) implementation with drag'n'drop.
https://github.com/nicolaskruchten/pivottable

check_graphite

Nagios plugin to poll Graphite for data.
https://github.com/datacratic/check_graphite
